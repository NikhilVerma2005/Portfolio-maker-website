# 🧩 Portfolio Maker Website

A dynamic and user-friendly web application built with **Flask** that allows users to create beautiful, responsive personal portfolios in just a few clicks. Select a design, fill in your details, and get a personalized portfolio ready to share!

## ✨ Features

- 🎨 Multiple pre-designed, modern portfolio templates
- 🧾 Step-by-step form to input personal, educational, and professional details
- 📷 Profile picture upload with auto-adjustment
- 🔗 Social media links integration (GitHub, LinkedIn, Instagram, Email)
- 📁 Automatically generates a shareable portfolio page
- 📱 Fully responsive and mobile-friendly

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- Python 3.x
- Flask
- Bootstrap (included via CDN)

### Installation

```bash
git clone https://github.com/your-username/portfolio-maker.git
cd portfolio-maker
pip install -r requirements.txt
python app.py
